Linear expressions and relations
[Direct modelling]

Collaboration diagram for Linear expressions and relations:

Detailed Description

Linear expressions can be freely composed of sums and differences of integer variables (Gecode::IntVar) possibly with integer coefficients and integer constants.

Note that integer variables are automatically available as linear expressions.

Linear relations are obtained from linear expressions with the normal relation operators.
