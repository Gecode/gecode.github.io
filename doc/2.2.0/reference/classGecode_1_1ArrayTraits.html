Gecode::ArrayTraits

Gecode::ArrayTraits< A > Class Template Reference

Detailed Description

template<class A>
class Gecode::ArrayTraits< A >

Traits of arrays in Gecode.

This class collects the traits of an array in Gecode. The traits used are the following.

  • typedef Type storage_type where Type is the type of an appropriate storage type for this array.
  • typedef Type value_type where Type is the type of the elements of this array.
  • typedef Type args_type where Type is the type of the appropriate Args-array type (e.g., BoolVarArgs if A is BoolVarArray).
