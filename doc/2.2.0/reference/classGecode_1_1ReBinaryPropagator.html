Detailed Description

template<class View, PropCond pc, class CtrlView>
class Gecode::ReBinaryPropagator< View, pc, CtrlView >

Reified binary propagator.

Stores two views of type View with propagation condition pc and a Boolean control view of type CtrlView.

If the propagation condition pc has the value PC_GEN_NONE, no subscriptions are created for View.
