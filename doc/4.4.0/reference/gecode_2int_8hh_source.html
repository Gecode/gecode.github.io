02522   /* \brief Post propagator for multi-dimensional bin packing
02523    *
02524    * In the following \a n refers to the number of items and \a m
02525    * refers to the number of bins.
02526    *
02527    * The multi-dimensional bin-packing constraint enforces that
02528    * all items are packed into bins
02529    * \f$b_i\in\{0,\ldots,m-1\}\f$ for \f$0\leq i<n\f$
02530    * and that the load of each bin corresponds to the items
02531    * packed into it for each dimension \f$l_{j\cdot
02532    * d + k} = \sum_{\{i\in\{0,\ldots,n-1\}|
02533    * b_{j\cdot d+k}=i}\}s_{i\cdot d+k}\f$
02534    * for \f$0\leq j<m\f$, \f$0\leq k<d\f$
02535    * Furthermore, the load variables must satisfy the capacity
02536    * constraints \f$l_{j\cdot d + k} \leq
02537    * c_k\f$ for \f$0\leq j<m\f$, \f$0\leq k<d\f$.
02538    *
02539    * The constraint is implemented by the decomposition
02540    * introduced in: Stefano Gualandi and Michele Lombardi. A
02541    * simple and effective decomposition for the multidimensional
02542    * binpacking constraint. CP 2013, pages 356--364.
02543    *
02544    * Posting the constraint returns a maximal set containing conflicting 
02545    * items that require pairwise different bins.
02546    *
02547    * Note that posting the constraint has exponential complexity in the
02548    * number of items due to the Bron-Kerbosch algorithm used for finding
02549    * the maximal conflict item sets.
02550    *
02551    * Throws the following exceptions:
02552    *  - Of type Int::ArgumentSizeMismatch if any of the following properties
02553    *    is violated: \f$|b|=n\f$, \f$|l|=m\cdot d\f$, \f$|s|=n\cdot d\f$,
02554    *    and \f$|c|=d\f$.
02555    *  - Of type Int::ArgumentSame if \a l and \a b share unassigned variables.
02556    *  - Of type Int::OutOfLimits if \a s or \a c contains a negative number.
02557    * 
02558    * \ingroup TaskModelIntBinPacking
02559    */

02561   binpacking(Home home, int d,
02562              const IntVarArgs& l, const IntVarArgs& b, 
02563              const IntArgs& s, const IntArgs& c,
02564              IntConLevel icl=ICL_DEF);
