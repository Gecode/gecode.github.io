01668   /*
01669    * \brief Relaxed assignment of variables in \a x from values in \a sx
01670    *
01671    * The variables in \a x are assigned values from the assigned variables
01672    * in the solution \a sx with a relaxation probability \a p. That is,
01673    * if \$fp=0.1\f$ approximately 10% of the variables in \a x will be
01674    * assigned a value from \a sx.
01675    *
01676    * The random numbers are generated from the generator \a r. At least
01677    * one variable will not be assigned: in case the relaxation attempt
01678    * would suggest that all variables should be assigned, a single
01679    * variable will be selected randomly to remain unassigned.
01680    *
01681    * Throws an exception of type Set::ArgumentSizeMismatch, if \a x and
01682    * \a sx are of different size.
01683    *
01684    * Throws an exception of type Set::OutOfLimits, if \a p is not between
01685    * \a 0.0 and \a 1.0.
01686    *
01687    * \ingroup TaskModeSet
01688    */

01690   relax(Home home, const SetVarArgs& x, const SetVarArgs& sx,
