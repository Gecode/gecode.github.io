                 Gecode 1.3.0
            http://www.gecode.org/

new infrastructure making it easier to add new variable domains. In  
addition, it contains recent bug fixes and minor improvements.

For a complete changelog, see
http://www.gecode.org/gecode-doc-latest/PageChanges_1_3_0.html

Cheers,
	The Gecode Team
