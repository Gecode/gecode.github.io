Hi Malcolm,

at least on Windows you can set maximum Java heap size using the "-Xmx" 
command line option, e.g.

java -Xmx512 ...

sets the limit to 512MB. I think that the default is set pretty low 
sometimes, so you might want to try this.

Hope this helps,
