Hi,

to make the HTML API a bit more readable and structured I would suggest 
to highlight each function description header such that a fast 
scolling/browsing is possible.

For that I suggest the following tag to the documentation css that draws 
a light grey box for the according div element

================================================
div.memproto {
	background-color: #FAFAFA;
	border: 1px solid #E0E0E0;
}
================================================

Maybe give it a try. :)
