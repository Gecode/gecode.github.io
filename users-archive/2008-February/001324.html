We are proud to announce the release of


                  QeCode 1.2 for Gecode 2.0.1

http://www.univ-orleans.fr/lifo/software/qecode/


QeCode is a solver for Quantified Constraint Satisfaction Problems
(QCSP) built on top of Gecode.  All constraints available in Gecode can
be used in quantified problems.  QeCode implements an extension of pure
QCSP called QCSP+ which allows to model problems in a natural way using
restricted quantification.  A short introduction to QeCode programming
is provided in the project home page.

QeCode 1.2 requires Gecode 2.0.1 and can be run on Windows, Linux
and Mac OS X.  Source code is provided as a contribution available in  
the
Gecode archive.

This software is released under MIT licence.
