Hi,

In addition to Guido's comments, you should be aware that the
SizeOptions class is defined in the support files in the examples
directory. These are not in the library files. To use the examle
support, you must also link it to the following object files:
    examples/support.o
    examples/timer.o
    examples/support/example.o
    examples/support/options.o

Cheers,
Mikael Lagerkvist
