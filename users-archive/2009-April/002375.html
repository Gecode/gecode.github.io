Actually, for 2. this is used: suppose you want express logical equivalence
by reification, say (x<y) ó (a < b). Then, one could express it by (x < y) ó
b0 = 1, (a < b) ó b1 = 1, and eqv(b0,b1,1). However it can be done more
efficiently by using just a single Boolean variable: (x < y) ó b=1 and (a <
b) ó 1. Only one Boolean variable, and no equivalence propagator.
