Hi Gustavo,

sorry, the tutorial documentation was wrong.  There is no support for  
posting n-ary set operations with arbitrary relations.  You have to  
decompose by hand using an auxiliary variable.  Thanks for pointing  
this out!

Cheers,
	Guido
