Element is just array access where the index and result are variables.

 

However, I think you got deceived by IntArgs: The first argument tells how
many values are to come. That is, when you write

                IntArgs cf(4, 7,4,9,11);

It will create an integer argument array with four elements which are
{7,4,9,11}.
