The argument to the linear constraint (*.col(a)) defines the coefficients
for the variables in the second argument  (*.row(d)). They  must be of the
same size (same number of arguments), naturally. If not, an exception is
thrown. 

 

Check the reference documentation of linear, it says in which cases linear
throws which exceptions.
