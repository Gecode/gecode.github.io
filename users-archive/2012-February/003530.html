Hi Alexis,

 

Yep, I know ILOG CP reasonably well. But I have to admit it is really clear
what you are trying to achieve.

 

If you just want to solve goals that are a conjunction that corresponds to
posting the respective goals inside a space. 

 

As you say yourself, disjunctive goals corresponds to a choice generated by
a brancher. Posting a new brancher inside the commit function of some other
brancher is possible in fact. I am not really sure what you mean by the
return type, but let me try: posting a brancher does not fail. So what you
can do inside the commit function is to post a new brancher and return ES_OK
as ExecStatus (signalling that the commit indeed did not fail, a commit
could fail if it for example posted a new constraint).

 

I hope that helps somewhat.
