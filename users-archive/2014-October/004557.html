Dear all,

we are releasing the next version of Gecode:

                      Gecode 4.3.1
             http://www.gecode.org

This release fixes some minor issues and a major bug for the FlatZinc interpreter. It considerably extends restart-based search so that it supports LNS (Large Neighborhood Search).

On our web site, you find source packages, binary packages for Windows and Mac OS, and the full documentation.

Cheers,
