<p class="MsoNormal">&lt;Description&gt;<br>
<span style="" lang="EN-GB">We are
currently trying to solve a problem using gecode, but cant quite get it to work
the way we want it to.</span></p>


<p class="MsoNormal"><span style="" lang="EN-GB">&nbsp;</span></p>






<p class="MsoNormal"><span style="" lang="EN-GB">&lt;Problem
description&gt;<br>
For every
column in the matrix, all choices must be present once. Not all rows need to
contain a choice, but for every column all choices must be found in one of the
rows. For every row, there is a corresponding constant in constants. The sum of
all choices in a row divided by the constant is put in percentage. For the
entire matrix, we want to find the distribution of choices in the matrix so
that percentage is as even as possible for all rows.</span></p>


<p class="MsoNormal"><span style="" lang="EN-GB">&nbsp;</span></p>




<p class="MsoNormal"><span style="" lang="EN-GB">&lt;What we
can't get to work&gt;<br>
We need
some help with how to setup this problem. Currently we are branching on x and
while that does give us a valid solution, its not really the one we are looking
for. Ideally we would like to find the most balanced solution (as determined by
the deviation of percentage for every row in comparison to percentage for the
matrix as a whole), not just a solution as we are currently.</span></p>


<p class="MsoNormal" style=""><span style="color: black;" lang="EN-GB">&nbsp;</span></p>


<p class="MsoNormal" style=""><span style="color: black;" lang="EN-GB">If every row in the
matrix consists of:</span></p>


<p class="MsoNormal" style=""><span style="color: black;" lang="EN-GB">Xi1 * A1 + Xi2 * A2 +
... + Xin * An + Ri = row_max[i]</span></p>


<p class="MsoNormal" style=""><span style="color: black;" lang="EN-GB">&nbsp;</span></p>


<p class="MsoNormal"><span style="color: black;" lang="EN-GB">We
want the smallest possible deviation of<span style="">&nbsp;
</span>Ri / row_max[i] for all i in the row_max array.</span><span style="" lang="EN-GB"></span></p>


<p class="MsoNormal"><span style="" lang="EN-GB">&nbsp;</span></p>


<p class="MsoNormal"><span style="" lang="EN-GB">There are
currently two problems facing us. For one, we are unsure of what relations to
enter for percentage, and secondly we are unsure on how to make gecode pick x
for the matrix based on values in percentage.</span></p>
