Hello Gecode Comunity !<br><br>Together with the AVISPA research group we are trying to develop<br>
a real time interpreter for a process calculus called <br>Non deterministic Timed Concurrent Constraint (NTCC), this calculus is a <br>non deterministic extension to the Timed Concurrent Constraint Calculus (TCC).<br><br>
In the past, some implementations were done using  Mozart OZ using combinators, but the
<br>real time was not achieved. We are now considering using GECODE with <br>reification, but the whole concurrency syncronization seems to be a big problem, additionally<br>reification could be not general enough.<br><br>
