domain from one space to another. As said, it can be done but it is non-trivial:
it would require locking as different spaces can be run by different threads.<o:p></o:p></span></p>

propagator? Should it be cloned as well?<o:p></o:p></span></p>

basically be two propagators, one in each space.<o:p></o:p></span></p>
