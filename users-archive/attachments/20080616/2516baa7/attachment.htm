An IntSet is never modified by a call to dom, after all, it&#39;s not a constraint variable! &nbsp;It works exactly the other way around: the variable domain is constrained to be a subset of the IntSet.<br>

The code is not tested, but this is approximately how it should work:<br>
<br>
void copythedomain(Space* home, IntVar from, IntVar to) {<br>
 &nbsp;IntVarRanges fromR(from);<br>
 &nbsp;IntSet fromS(fromR);<br>
 &nbsp;dom(home, to, fromS);<br>
}<br>
<br>
Note that home has to be the space that &quot;to&quot; lives in, and you don&#39;t need the space that &quot;from&quot; lives in.<br>
