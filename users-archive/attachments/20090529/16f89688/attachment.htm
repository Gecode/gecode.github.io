<p class=MsoNormal><span style='font-size:11.0pt;font-family:"Calibri","sans-serif";
color:#1F497D'>No operation in Gecode is thread safe (by design). So, when you
want to create a clone of a space, only one thread can do that at a time (and
you have make sure that clone is called with false as argument for share). If that
turns out to be a bottleneck for your application: if you have created an
unshared clone c from a space s, then of course two threads can create clones
in parallel: one from c and one from s.<o:p></o:p></span></p>
