<p class=MsoNormal><span style='font-size:11.0pt;font-family:"Calibri","sans-serif";
color:#1F497D'>Nope, MwG is correct on that respect. &#8220;x &amp;&amp; imp(y,z)&#8221; is
a Boolean expression, whereas &#8220;a[j] == i+1&#8221; is a relation (constraint). Posting
a Boolean expression yields a Boolean variable, posting a relation does not.
Please check the definition of expressions and relations in MwG.<o:p></o:p></span></p>
