<p class=MsoNormal><span style='font-size:11.0pt;font-family:"Calibri","sans-serif";
color:#1F497D'>The argument to the linear constraint (*.col(a)) defines the coefficients
for the variables in the second argument&nbsp; (*.row(d)). They &nbsp;must be of the same
size (same number of arguments), naturally. If not, an exception is thrown. <o:p></o:p></span></p>

<p class=MsoNormal><span style='font-size:11.0pt;font-family:"Calibri","sans-serif";
color:#1F497D'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='font-size:11.0pt;font-family:"Calibri","sans-serif";
color:#1F497D'>Check the reference documentation of linear, it says in which
cases linear throws which exceptions.<o:p></o:p></span></p>
