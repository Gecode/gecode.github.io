<div class="im">Let&#39;s assume the IntVarArray is called x, and we want at most max unique elements.</div>
<br>
There are two possible ways to model this:<br>
(1) Use an additional IntVarArray y and a count constraint so that y[i] is the number of occurrences of the ith possible element. Then use reification to get Boolean variables b[i] &lt;-&gt; y[i] &gt; 0, and the number of unique elements will be the sum of the Booleans, which you can constrain to be &lt; max.<br>

</blockquote><div><br></div><div>There is a variant of this that has the same propagation, but it uses less constraints and variables so it might be a bit better. </div><div><br></div><div>Lets say that you have n variables and that they use k different values. If you count the number of zeroes in the y-array (one count constraint), you get a value z which is equal to n-k. To constrain the maximum number of values, just make sure that there is a minumum number of zeroes: z &gt;= n-max.</div>
