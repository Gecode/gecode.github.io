am struggling with finding the best way to cut a hole in a view during
propegation without doing a simple for loop. I originally thought I should use
a Range and Iterator and execute the minus_v method, but I can’t find an
easy way to construct one easily using specific values other than using a value
array of ints but I would still need a loop to initialize the array. What is
the best way to do this with the least computation?  </p>

<p class="MsoNormal"> </p>

<p class="MsoNormal">The
end goal of this propagator is to put a gap between two domains however there
is no guarantee that x0 &lt; x1  so you really can’t reduce much on
the domain until one is assigned a start. Additionally you can’t use an
offset view because you only want the offset to apply if the task is the
earlier one one.</p>

<p class="MsoNormal" style="text-autospace:none"><span style="font-size:10.0pt;font-family:&quot;Courier New&quot;">

<p class="MsoNormal" style="text-autospace:none"><span style="font-size:10.0pt;font-family:&quot;Courier New&quot;">                 
}</span></p>

<span style="color:blue">else</span> <span style="color:blue">if</span>
(x1.assigned())</span></p>

<p class="MsoNormal" style="text-autospace:none"><span style="font-size:10.0pt;font-family:&quot;Courier New&quot;">

<p class="MsoNormal" style="text-autospace:none"><span style="font-size:10.0pt;font-family:&quot;Courier New&quot;">                 
<span style="color:green">//create a range that x0 cant equal over (x1.val -

<p class="MsoNormal" style="text-autospace:none"><span style="font-size:10.0pt;font-family:&quot;Courier New&quot;">                 
<span style="color:blue">for</span> (<span style="color:blue">int</span> i =
x1.val() - offset0; i &lt; x1.val() + offset1; ++i)</span></p>

<p class="MsoNormal" style="text-autospace:none"><span style="font-size:10.0pt;font-family:&quot;Courier New&quot;">
