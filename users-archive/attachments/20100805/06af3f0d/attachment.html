Hi Kish,<div><br></div><div>Yes, the API has changed significantly between 3.0.2 and 3.4.0. The 3.4.0 release in particular has had a lot of changes in conjunction with writing Modeling and Programming with Gecode to make the interfaces consistent and easier to explain. Our version-number strategy is documented here: <a href="http://www.gecode.org/changes.html">http://www.gecode.org/changes.html</a> All relevant API-changes are documented in the changelog for the version they are introduced for.</div>

<div><br></div><div>As for the init function, it was removed since it is not needed. If you wrote</div><div> a.init(*this, 0, 10);</div><div>with a an IntVar, you can replace it with</div><div> a = IntVar(*this, 0, 10);<br>
