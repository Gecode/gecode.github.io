Given your example, I think you need something like this:<br>
<br>
BoolVarArgs _m(home, n_tasks*n_resources);<br>
Matrix&lt;BoolVarArgs&gt; m(m, n_tasks, n_resources);<br>
<br>
for (int i=0; i&lt;n_resources; i++) {<br>
  for (int j=0; j&lt;n_tasks; j++) {<br>
    rel(home, m(j,i)  == (resource[j] == i));<br>
    dom(home, start_time[j], set_possible_values, m(i,j));<br>
  }<br>
  unary(home, startTimes, durations, m.row(i));<br>
}<br>
for (int i=0; i&lt;n_tasks; i++) {<br>
  linear(home, m.col(i), IRT_EQ, 1); // each task is scheduled on exactly one resource<br>
}<br>
<br>
For the m(j,i)  == (resource[j] == i) constraint, you can actually use a channel constraint, which is more efficient, but I thought this is closer to your original code.<br>
