Hi Manel,<div><br></div><div>I think you want something that looks like this:</div><div><br></div><div>int n = 8;</div><div>int m = 10;</div><div>IntVarArray a(*this,n*m,0,1); // Array of n*m boolean variables</div><div>Matrix<IntVarArray> mat(x,n,m); // Matrix "view" of the array a</div>
<div><br>Now, to post a constraint on every row in the matrix you can use the method row of the Matrix class. That will return you exactly what you need. For instance, to post that the row i has pairwise distinct values:</div>
<div><br></div><div>distinct(*this,mat.row(i));</div><div><br></div><div>Be careful, that is just an example that is (probably) not going to work because of the use of boolean variables. In any case you will find more information on section 7.2 in MPG about using Matrix.</div>
