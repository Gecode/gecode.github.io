We are happy to announce that Gecode 5.0.0 has been released on October 25,<br>
2016. Here is an excerpt from the changelog:<br>
<br>
This release adds a number of new features and fixes many bugs (some rather<br>
serious), so it is recommended to switch as soon as possible to this<br>
release. Some highlights are: re-added limited discrepancy search (LDS);<br>
added sequential and parallel portfolio-based search (PBS); added support<br>
for groups of propagators and branchers; added extensive tracing<br>
functionality based on groups; propagators can be disabled and re-enabled;<br>
support for arbitrary memory allocators; changed the notion of "integer<br>
consistency level" to the more expressive concept "integer propagation<br>
level".<br>
